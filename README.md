# Blogicum

**Blogicum** — это веб-приложение блога, разработанное на Django. Пользователи могут публиковать записи, редактировать их, добавлять комментарии, просматривать профили авторов и управлять своим контентом.

Проект реализует полноценный блог с системой пользователей, категориями, изображениями и административной панелью.

---

## Функциональность

Приложение поддерживает:

* регистрацию и авторизацию пользователей
* создание, редактирование и удаление постов
* загрузку изображений к постам
* просмотр всех опубликованных постов
* просмотр постов по категориям
* просмотр профиля пользователя
* добавление и удаление комментариев
* пагинацию списка постов
* административную панель Django
* обработку пользовательских прав доступа

Только автор может редактировать и удалять свои посты и комментарии.

---

## Технологии

Проект использует:

* Python 3.10+
* Django
* SQLite
* Django ORM
* Django Templates
* Bootstrap
* pytest (для тестирования)

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <your-repository-url>
cd django_sprint4/blogicum
```

---

### 2. Создать виртуальное окружение

```bash
python -m venv venv
```

Активировать:

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

---

### 3. Установить зависимости

```bash
pip install -r ../requirements.txt
```

---

### 4. Применить миграции

```bash
python manage.py migrate
```

---

### 5. Создать администратора

```bash
python manage.py createsuperuser
```

---

### 6. Запустить сервер

```bash
python manage.py runserver
```

Сайт будет доступен по адресу:

```
http://127.0.0.1:8000/
```

Административная панель:

```
http://127.0.0.1:8000/admin/
```

---

## Административная панель

Администратор может:

* управлять постами
* управлять категориями
* управлять пользователями
* модерировать комментарии
